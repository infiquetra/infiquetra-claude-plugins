# Changelog

All notable changes to the Infiquetra Documentation Generator plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-12

### Added
- Initial release as part of Infiquetra Plugin Marketplace
- OpenAPI specification generation from Lambda handlers
- Architecture diagram generation using Python diagrams
- README template generation with Infiquetra standards
- Integration guide creation with code examples
- Sequence diagram generation using mermaid
- Cross-reference management across documentation
- Automatic documentation validation

### Features
- **Generate Command**: Create all documentation types
- **Validate Command**: Verify documentation completeness and accuracy
- **Publish Command**: Deploy documentation to GitHub Pages
- **Template System**: Infiquetra-standard documentation templates
- **Diagram Support**: Architecture and sequence diagrams

### Documentation Types
- API specifications (OpenAPI/Swagger)
- Architecture diagrams (PNG format)
- README files (Markdown)
- Integration guides (with examples)
- Sequence diagrams (Mermaid)

### Time Savings
- 30-45 minutes for documentation projects
- Standardized documentation across Infiquetra services
- Automated cross-reference management
