# Facebook Marketplace Fields Reference

All fields, categories, condition values, and tips for creating effective listings.

## Required Fields

| Field | Limit | Notes |
|-------|-------|-------|
| **Title** | 100 characters | Most searchable terms first |
| **Price** | — | Set to 0 for "Make Offer" |
| **Category** | — | See category list below |
| **Condition** | — | Must use exact values |
| **Location** | — | Auto-fills from profile |
| **Description** | 5,000 characters | Markdown not supported |

## Title Optimization

**Format:** `[Brand] [Model] [Key Spec] - [Condition]`

Rules:
- Put most searchable terms first (buyers scan titles)
- Include model number for electronics
- Include size for furniture
- Include capacity for appliances
- "Firm" reduces lowball offers ~40%

**Examples:**
- `DeWalt 20V MAX Drill Driver Kit w/ 2 Batteries - Like New`
- `IKEA MALM Queen Bed Frame White - Good Condition`
- `iPhone 14 Pro 256GB Space Black Unlocked - Excellent`
- `Vitamix 5200 Blender - Good, All Parts Included`
- `Trek FX3 Hybrid Bike 54cm - Good Condition`

## Condition Values

Use exactly as written:

| Value | Meaning |
|-------|---------|
| `New` | Never used, still in original packaging |
| `Like New` | Used once or twice, no signs of wear |
| `Good` | Normal use, fully functional, minor cosmetic issues |
| `Fair` | Significant wear but functional, cosmetic issues |
| `Poor` | Major issues, for parts or project |

## Description Template

```
[What it is and why someone wants it — 1-2 sentences that lead with value]

Key specs:
- [Spec 1]
- [Spec 2]
- [Spec 3]

Condition: [Condition level]. [Honest detail about any wear, scratches, or issues].

What's included: [List everything in the box/sale]

Cash or Venmo. Pickup in [neighborhood], Indianapolis. No holds without deposit.
```

**Description Tips:**
- Lead with the benefit, not the feature ("Keeps drinks cold 48 hours" > "40 oz insulated tumbler")
- Be honest about defects — buyers will ask anyway, and it builds trust
- Specify pickup logistics upfront to save back-and-forth
- "No holds without deposit" filters out time-wasters
- Mention if item is smoke-free / pet-free home (matters to many buyers)

## Photo Requirements

Facebook allows up to 10 photos. Order matters — first photo is the listing thumbnail.

**Optimal photo order:**
1. Best overall shot (clean background, good light) — this is the thumbnail
2. Second angle
3. Close-up of model/brand label
4. Any defects or wear (builds trust)
5. What's included (all accessories laid out)
6. Size reference if non-obvious

**Technical:**
- Minimum: 500 × 500px
- Recommended: 1080 × 1080px or higher
- Square or landscape (portrait gets cropped)
- HEIC uploads work on iOS/macOS; JPEG more universal

## Listing Tips

**Timing:**
- Post Thursday–Sunday for maximum weekend traffic
- Avoid posting Monday–Wednesday (lowest buyer activity)
- Electronics sell well any time; furniture moves best spring/fall

**Pricing:**
- List 10-15% above your minimum to leave room for offers
- Don't lower price immediately — first 48 hours are highest traffic
- "OBO" (Or Best Offer) invites lowballs; "Firm" on good-condition items performs better

**Responding to buyers:**
- Reply within 1 hour during business hours (FB surfaces responsive sellers)
- Meet in public for high-value items (police station parking lots are common)
- Venmo/cash on pickup — no shipping, no payment plans

**Cross-listing:**
If no sale in 2 weeks, expand to:
- **eBay** — electronics, collectibles, brand-name tools (national market)
- **Craigslist** — large items like furniture, appliances (local-only buyers)
- **Facebook Groups** — specialty groups for hobbies (fishing, cycling, gaming)
