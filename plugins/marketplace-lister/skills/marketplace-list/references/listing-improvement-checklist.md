# Listing Improvement Checklist

Use this checklist after generating an initial listing but before saving. Present only actionable suggestions — skip items where the listing is already good.

## Photo Review

Check each of these against the photos you viewed:

### Coverage
- [ ] **Hero shot** — Is the first photo the best overall angle? (move if not)
- [ ] **All major angles** — Front, back, sides, top covered?
- [ ] **Ports/connections** — For electronics: are ports, inputs, and outputs visible?
- [ ] **Model/serial label** — Close-up of model sticker or serial number?
- [ ] **Defects documented** — Any visible damage shown in photos? (builds trust, reduces surprises)
- [ ] **Accessories shown** — Are all included items in at least one photo?
- [ ] **Scale reference** — Something in frame to show how big the item is?

### Quality
- [ ] **Lighting** — Are photos bright enough? Dark/shadowy photos reduce views
- [ ] **Background** — Neutral/uncluttered background preferred (floor, wall, table)
- [ ] **Sharpness** — Are key detail shots in focus?

### Mercari-Specific
- [ ] **Item powered on** — For electronics: does any photo show it powered on / LEDs lit?
- [ ] **12 photo limit** — Could more photos help? (Mercari allows 12)

### Suggested Photo Order
1. Best overall angle (hero)
2. Second major angle
3. Model label close-up
4. Any defects or wear
5. Included accessories
6. Scale reference
7. Additional detail shots

## Title Review

### FB Marketplace (100 char limit)
- [ ] Includes brand + model + key spec?
- [ ] Condition mentioned (or implied by "used")?
- [ ] Avoids filler words ("nice", "great", "barely used")?
- [ ] "Firm" included if you don't want to negotiate?

### Mercari (80 char limit)
- [ ] Keyword-dense — uses buyer search terms?
- [ ] Brand + model + key spec packed in?
- [ ] Every character earns its place?

## Description Review

### Content Completeness
- [ ] What it is and why someone wants it (opening hook)?
- [ ] All key specs that buyers search for or ask about?
- [ ] Condition details that preempt questions ("no cracks", "works perfectly", "one dead pixel")?
- [ ] Accessories/contents list (exactly what's included)?
- [ ] Pickup/payment logistics at the end?

### Buyer Psychology
- [ ] Any "social proof" applicable? (popular model, well-reviewed, known brand)?
- [ ] Compatibility mentioned for electronics? (what systems, OS, standards it works with)?
- [ ] Any setup/configuration notes? (some buyers worry about used electronics being hassle)

### Mercari-Specific
- [ ] 1–3 relevant hashtags at the end? (#brand #category #keyword)
- [ ] Stays under ~800 characters so it shows fully without "see more"?

## Pricing Review

### Market Alignment
- [ ] Does pricing match eBay sold comps? (note any significant difference)
- [ ] Does pricing match local FB comps? (Indianapolis runs 10–15% below coastal)
- [ ] Is there a seasonal factor? (coolers in fall, heaters in spring — harder sells)

### Opportunity
- [ ] Bundle opportunity with other items in inventory? (same-brand, complementary items)
- [ ] Could quick-sale price move this faster if the user wants quick cash?
- [ ] Is Mercari price correctly 11% above FB to cover fees?

## Platform Selection Review

- [ ] Is platform recommendation appropriate for item size/weight?
- [ ] Is ship-vs-local call correct for the weight and price?
- [ ] If shipping, is the carrier/cost estimate reasonable?

## Presenting Suggestions

Format suggestions grouped by category. Only include categories with actual suggestions:

```
## Suggested Improvements

Before I save this listing, here are a few suggestions:

📸 **Photos:**
1. [Specific suggestion]
2. [Specific suggestion]

✏️ **Title/Description:**
1. [Specific suggestion]

💰 **Pricing:**
1. [Specific suggestion]

Apply these improvements? [Yes, all / Pick which ones / Skip]
```

Skip categories with nothing to add. Keep suggestions specific and actionable — not "take better photos" but "add a photo showing the rear ports."
