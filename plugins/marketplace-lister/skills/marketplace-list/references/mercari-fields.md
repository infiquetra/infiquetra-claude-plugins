# Mercari Listing Fields Reference

## Listing Fields

| Field | Limit | Notes |
|-------|-------|-------|
| **Title** | 80 characters | Keyword density matters — buyers search by keyword |
| **Description** | 1,000 characters | Up to 3 hashtags allowed (at end of description) |
| **Category** | Hierarchical | Navigate to most specific subcategory |
| **Condition** | Enum (5 values) | See below |
| **Brand** | Free text | Required for electronics/clothing |
| **Color** | Dropdown | Optional but helps search |
| **Photos** | Up to 12 | First photo is thumbnail |
| **Price** | $1–$2,000 | Buyer can make offers if you enable it |
| **Shipping** | Prepaid label or local | See Shipping section below |

## Condition Values

| Value | Use When |
|-------|----------|
| New | Never used, original packaging preferred |
| Like New | Used once or twice, no visible wear |
| Good | Normal use, fully functional, minor cosmetic wear |
| Fair | Significant cosmetic wear but fully functional |
| Poor | Major wear or cosmetic issues, still functional |

## Fee Structure (as of Jan 2025)

- **Selling fee:** 10% of (item price + buyer-paid shipping)
- **Payment processing fee:** $0 (eliminated January 2025)
- **Payout:** Free direct deposit, 3 business days after delivery

### Mercari Price Formula

To net the same proceeds as Facebook Marketplace (which is free), price Mercari listings 11-12% higher:

```
Mercari Price = FB Price ÷ 0.90
```

Example: FB fair market = $70 → Mercari price = $70 ÷ 0.90 = **$78**

## Shipping Options

### Prepaid Label (recommended)

Mercari provides discounted prepaid labels through USPS, UPS, and FedEx. Rates are ~54% off retail.

| Service | Weight Tier | Approx. Cost |
|---------|-------------|--------------|
| USPS First Class | Under 1 lb | $3.50–5.50 |
| USPS Ground Advantage | 1–10 lbs | $6–14 |
| UPS Ground (Best Rate) | Under 1 cu ft, under 22" | 6% cheaper than Standard |
| UPS Ground (Standard) | Any | Varies by weight/zone |
| FedEx Ground | Any | Comparable to UPS |

**Current promo (Feb–Apr 2026):**
- UPS/FedEx 1–5 lb: $4.99 flat
- USPS First Class Envelopes: $0.49

**Overweight penalty:** 8% surcharge if actual weight exceeds label weight.

### Ship vs. Local Decision

- **Ship if:** Item weighs < 10 lbs AND price > $25 (shipping economics make sense)
- **Local only if:** Item is large/heavy, fragile, or price < $20

## Mercari-Specific Listing Tips

### Title Optimization

Mercari search is keyword-driven. Pack the 80-character title with searchable terms:
- Include brand + model + key spec + condition
- Use buyer vocabulary, not technical jargon
- Skip words like "used", "great", "nice" — they waste characters

**FB title:** `YETI Tundra 45 Cooler White - Good Condition`
**Mercari title:** `YETI Tundra 45 Hard Cooler White 45qt Good`

### Description Tips

- Lead with specs (buyers scan fast)
- Use bullet-point style even without markdown (Mercari doesn't render markdown)
- End with 1–3 relevant hashtags: `#yeti #cooler #camping`
- Keep under 800 characters so it shows fully without "see more"

### Mercari AI Listing

Mercari's app has an AI listing feature that generates a listing from photos in 3 taps. It's decent for common items. Use it as a starting point and refine — it often misses specs and uses generic descriptions.

### Buyer Communication

- Respond quickly — response time affects search ranking
- Be factual about condition — Mercari's buyer protection is strict
- Ship within 3 days of purchase (required by Mercari policy)
