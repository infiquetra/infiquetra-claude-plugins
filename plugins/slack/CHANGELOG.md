# Changelog

## [1.0.0] - 2026-02-26

### Added
- Initial release of slack plugin
- **slack-messaging** skill: Send messages, search, threads, reactions
- **slack-channels** skill: Channel management and history
- `slack_client.py`: Slack Web API client (~500 lines)
- Bot token support for posting
- User token support for searching
- Integration with PagerDuty for incident notifications
