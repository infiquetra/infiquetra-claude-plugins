# Infiquetra Slack Plugin

Slack messaging and channel management for Infiquetra team communication.

## Features

- Send messages and thread replies
- Search message history
- Channel management (list, create, archive)
- User lookup
- Reactions

## Installation

```bash
export SLACK_BOT_TOKEN="xoxb-your-bot-token"
export SLACK_USER_TOKEN="xoxp-your-user-token"  # Optional, for search
cp -r plugins/slack ~/.claude/plugins/
```

## Usage

```
Post to #team that deployment is complete
Search for incident discussions in Slack
Create incident war room channel
```
